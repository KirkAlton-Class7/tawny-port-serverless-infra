# Tawny Port Brand Identity

Use this quick sheet when applying the Cognito managed login look without reopening every console-level branding setting.

Provides:

* Primary color roles
* Secondary and utility colors
* Status colors
* Type color opacity guidance
* Cognito managed login defaults where typography is not specified

---

## Quick Direction

Luxury wine identity: deep port reds, midnight navy surfaces, champagne text, and restrained gold accents.

> [!NOTE]
> These colors come from the approved Cognito Managed Login setup. When uploading image assets, select the provided version that satisfies AWS Cognito file size requirements.

## Primary Palette

| Color | Hex | Use |
| --- | --- | --- |
| Midnight Navy | `#0F1B2A` | Main surface color for page, form, secondary controls, and dark UI grounding |
| Deep Port | `#570505` | Primary brand/action color for buttons, borders, and hover fills |
| Ruby Port | `#7A0B0B` | Elevated burgundy state for hover/active emphasis |
| Vintage Port | `#3D0303` | Deep pressed/active burgundy state |
| Champagne | `#F7E7CE` | Main text, labels, button text, field text, light fills, and soft borders |
| Warm Champagne | `#FFF4E3` | Brighter hover text and link hover color |
| Antique Gold | `#C9A227` | Focus state, selected controls, links, and premium accents |
| Bright Gold | `#F4D35E` | Highlight text for secondary button interaction states |

## Secondary / Utility Palette

| Color | Hex | Use |
| --- | --- | --- |
| Deep Navy Lift | `#192534` | Slightly lighter navy for active secondary surfaces |
| Tawny Divider | `#967C52` | Divider and subtle warm separation |
| Aged Gold | `#C7A12C` | Input border and refined metallic detail |
| White | `#FFFFFF` | Disabled primary button background and border |

## Status Palette

| Color | Hex | Use |
| --- | --- | --- |
| Success Green | `#3BCF57` | Success icon, text, and border |
| Success Dark | `#001A02` | Success background |
| Error Rose | `#F07C7C` | Error icon, text, and border |
| Error Dark | `#1A0000` | Error background |
| Pending Champagne | `#F7E7CE` | Pending state text at reduced opacity |

## Type Colors

| Type role | Color | Opacity |
| --- | --- | --- |
| Heading | `#F7E7CE` | `100%` |
| Description | `#F7E7CE` | `82%` |
| Body | `#F7E7CE` | `72%` |
| Field label/value | `#F7E7CE` | `84%` |
| Field description | `#F7E7CE` | `52%` |

## Opacity Notes

| Use | Color | Opacity |
| --- | --- | --- |
| Form background | `#0F1B2A` | `90%` |
| Page background | `#0F1B2A` | `100%` |
| Links | `#C9A227` | `80%` |
| Divider | `#967C52` | `80%` |
| Input placeholder | `#0F1B2A` | `92%` |
| Input background | `#F7E7CE` | `52%` |

| Pending status | `#F7E7CE` | `40%` |

## Type

Font family is not specified in the Cognito branding source. Use Cognito Managed Login defaults unless a separate Tawny Port brand font is provided.
